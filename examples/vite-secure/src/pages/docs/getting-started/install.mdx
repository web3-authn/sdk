# Install

Install the SDK in your app. Install the root package and import from subpaths like `@tatchi/sdk`, `@tatchi/sdk/react`, and `@tatchi/sdk/server`.

```bash
pnpm add @tatchi
# or
npm i @tatchi
# or
yarn add @tatchi
```

If you plan to use the Wallet Iframe components:

```bash
pnpm add lit react
```
